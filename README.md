# jquery-validate-eraser
> 가끔 요구사항 중에 input에서 이런저러한 값을 아예 입력을 하지 못하게 해달라고 하는
> 요구사항이 있는 경우가 있습니다. 매번 그러한 것을 위해 코드를 짜는 것도 귀찮고 해서
> jquery plugin 형태로 만들어서 사용중입니다.
